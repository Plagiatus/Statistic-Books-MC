Scoreboard Combiner

How to

Enter into the Input field(s) all the scoreboard objectives that you want to combine.
Enter into the Output field the scoreboard objective that you want them to be combined into.

Click either the "create function" button to get a function file or "create datapack" button to create an entire datapack with this function and a tag to work with the mcstats book tag.

WARNING: Only works out of the box with version 1.5.1 of the datapack or above! Get the latest release here.
